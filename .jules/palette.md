## 2026-06-28 - Keyboard Accessibility for Graph Nodes and Edges\n**Learning:** Graph elements (nodes and edges) need to be keyboard accessible to allow users to interact with them without a mouse. Adding tabindex='0' and handling 'Enter' and 'Space' keydown events is crucial for providing a fully inclusive experience.\n**Action:** Always ensure that interactive elements that are not native form controls or links are made focusable and handle keyboard events for activation.
